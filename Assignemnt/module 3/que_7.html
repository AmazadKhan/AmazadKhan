What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? With Example.

Ans: HTML id Attribute: The id attribute is a unique identifier that is used to specify the document.
It is used by CSS and JavaScript to perform a certain task for a unique element. In CSS, the id attribute
is written using the # symbol followed by id.

HTML class Attribute: The class attribute is used to specify one or more class names for an HTML element.
The class attribute can be used on any HTML element. The class name can be used by CSS and JavaScript to perform certain
tasks for elements with the specified class name. The class name in CSS stylesheet using “.” symbol.
