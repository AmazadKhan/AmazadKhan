Que 15) How to insert a picture into a background image of a web page? With Example.

Ans :To set the background image of a webpage, use the CSS style. Under the CSS < style > tag, add the property background-image. The property sets a graphic such as jpg, png, svg, gif, etc.
HTML5 do not support the < body > background attribute, so CSS is used to change set background image
