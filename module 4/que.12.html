Qu:-12 How to maintain the CSS specifications?

Answer:

* The CSS specifications maintained by the W3C
* This Css specifications also contain the syntax & deta type of the CSS style rule.
*Css specification provides detailed Explanations of the CSS selectors.
*The CSS Selectors are those,which selects the HTML elements ,on which we want to apply Various css style .
*For examples,Class selector maintained by "dot(.)" symbol in internal or external style sheet.
*Id selector maintained by "hash(#)" symbol in internal& external style sheet.
*Like above, Pseudoclass selector mainteined by "colon(:)" symbol in internal & external style sheet.
*Pseudoelement selector maintained by "Doublecolon(::)" symbol in internal & external style sheet.
